## SETUP IN LOCAL

- Clone from github
```bash
    https://github.com/frenzycoders/scpanel-ui.git
```

- Navigate to scpanel-ui and run 

```bash
    cd scpanel-ui && flutter pub get
```

- Well Done, Setup is completed now run 

```bash
    flutter run -d chrome
```
